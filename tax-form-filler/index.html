🔒 Privacy & Security

        
Your data never leaves your computer. This application runs entirely in your browser (client-side). No data is uploaded to any server. Your Excel file and PDFs are processed locally on your device.

        
💡 Tip: Your Excel file only needs numerical data (amounts, dates, quantities) - no personal information like names, addresses, or SSN is required.

📊 Excel Format for Form 8949

        
Your sheet should have these columns (in any order):

        
    
          
  • Quantity - Number of shares
    • 
          
  • Name - Security name
    • 
          
  • Purchase Date - Date acquired
    • 
          
  • Sell Date - Date sold
    • 
          
  • Purchase Price - Cost basis
    • 
          
  • Sell Price - Sale proceeds
    • 
          
  • Code - Adjustment code (optional, e.g., W for wash sale)
    • 
          
  • Adjustment - Adjustment amount (optional, default 0). Gain/Loss = Proceeds - Cost - Adjustment
    • 
        


        
📊 Excel Format for Form 1040

        
Two columns format:

        
    
          
  • Variable: Line numbers (e.g., "Line 1a", "Line 2a")
    • 
          
  • Value: Amounts
